# Angular CLI Guide

managing dependencies, configuring environments, and executing core development workflows.

## 1. System Prerequisites

### Install Node.js (via nvm)

> NOTE: The nvm first needed to install, checkout [nvm guide](https://github.com/Millstack/cli-playbook/blob/main/angular/nvm.md)

```bash
nvm install 20.0.0
nvm use 20.0.0

# note: to check the installed node and node package manager version
node -v
npm -v
```
